## Usage

List of using steps:
- [x] Clone the repository
    ```bash
    git clone https://github.com/IvanMao714/Plot_Academic_Block.git
    ```
- [x] run neturalnetwork.py
  ```bash
    python neturalnetwork.py
   ```
- [x] generate the latex file and compile it
  ```bash
    pdflatex output.tex
   ```
